# Personal Finance Tracker Pro

A modern expense tracking web application built using HTML, CSS, and JavaScript.

## Features

- Dashboard
- Add/Edit/Delete Transactions
- Income & Expense Tracking
- Search Transactions
- Category Filter
- Dark Mode
- Local Storage
- Chart.js Pie & Bar Charts
- PDF Export
- CSV Export
- Responsive Design

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- jsPDF

## How to Run

1. Download the project.
2. Open the folder in VS Code.
3. Open `index.html` in your browser.
4. Start managing your finances!
